Templates¶

Templates, i.e., “scaffold” XML/CSS content; these are finalized, and then added to the book, by the Package instance.

rp2epub.templates.meta_inf¶

XML code for the Manifest file

rp2epub.templates.COVER¶

XHTML scaffolding code for the cover page

rp2epub.templates.BOOK_CSS¶

Content of the book.css file, adding some page-breaking statements to the overall styling.
